ANTES DE "COMITAR" UM ARQUIVO, EXISTE ALGUNS PASSOS
ANTES;

PARA VERIFICAR SE HÁ ARQUIVOS QUE FORAM ALTERADOS,
EXECUTE O COMANDO:
- git status

AGORA PARA ADICIONAR UM ARQUIVO PARA O MODO "TRACK",
DIGITE:
- git add nome_do_arquivo

OBS: CASO VOCÊ FAÇA UMA MODIFICAÇÃO NO ARQUIVO, PARA
QUE ESSAS MODIFICAÇÕES SEJAM USADAS NO COMMIT, SERÁ
NECESSÁRIO USAR O COMANDO: git add nome_do_arquivo, 
NOVAMENTE.

APÓS FEITO TUDO ISSO, EXECUTE O COMANDO:
- git commit -m "mensagem que deseja passar"

FEITO ISSO, O GIT CRIARÁ UMA VERSÃO DO ARQUIVO, PODENDO
RESTAURA-LO E ENVIANDO PARA O GITHUB.
